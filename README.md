# admin
基于element-ui封装的一个适用与后台管理系统的骨架层 其中包含 主题更换、左侧及顶部菜单栏、界面自适应、axios 原型公共方法的封转（防抖、节流、精度计算、localStorage缓存等）
## Project setup
```
npm install
```
